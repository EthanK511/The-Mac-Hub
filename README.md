## Why I made it

I made this because I have issues with my MacBook. I don't have a place to plug in USB-A devices. 
This solves the problem and also fixes the issue that when I plug into an external monitor, I only have one USB-C port available

## Features

This has a very cool design, it's compact, and it has nice things printed on it.
It has 2 USB-C ports and 2 USB-A ports to plug things into.
